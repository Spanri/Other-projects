<venv> - путь и название окружения

Создать venv
python -m venv <venv>

Активировать venv
<venv>/Scripts/activate.ps1 - PowerShel
source <venv>/Scripts/activate - bash

Если что-то в питоне сломала
Сотри PYTHONHOME, PYTHONPATH (присвой им пустые значения)
(%pythonhome%)
C:\Program Files\Python\Python37\
(%path%)
C:\Program Files\Python\Python37\
(%pythonhome% если из виртуальной среды работать)
